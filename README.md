# CPC Website

This is the official repository for the CPC website.

## Installation

Recommended node-js version: `v14.15.4`

1. Clone this repo
2. Install dependencies: `npm install`
3. Run it: `npm start`

Live site: [https://cultoperrocafe.com](https://cultoperrocafe.com)
